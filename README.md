# LNear - Просмотр игроков поблизости

**LNear** — это простой и удобный плагин для вашего сервера Minecraft, который позволяет игрокам просматривать, кто
находится рядом с ними, с помощью команды `/near`.

## Особенности плагина:

* **Стрелки-указатели:** К каждому из найденных игроков показываются стрелки, куда идти чтобы их найти.
* **Поддержка зелья невидимости:** Игроки, находящиеся под действием эффекта невидимости, не будут отображаться в
  списке.
* **Настраиваемый радиус:** Можно гибко настроить разный радиус обзора для разных привилегий в конфиге.

## Команды:

* `/near` — Показать игроков поблизости.
* `/lnear reload` — Перезагрузить конфигурацию плагина.

## Права:

* `lnear.use` — Даёт доступ к использованию команды `/near`.
* `lnear.radius.<group>` — Устанавливает радиус поиска для определённой группы, где `<group>` — это название группы из
  конфига.
* `lnear.admin` — Даёт доступ к команде перезагрузки конфига `/lnear reload`.
